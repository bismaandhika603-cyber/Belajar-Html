# Belajar-Html
## 1. Mengenal HTML

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur sebuah halaman website.

Agar lebih mudah dipahami, bayangkan sebuah bangunan:

- HTML berfungsi sebagai kerangka bangunan.
- CSS berfungsi sebagai desain interior dan eksterior.
- JavaScript berfungsi sebagai sistem yang membuat berbagai fitur dapat berjalan dan berinteraksi dengan pengguna.

Tanpa HTML, sebuah website tidak memiliki struktur yang jelas.

---

## 2. Struktur Dasar HTML

Setiap halaman web umumnya diawali dengan struktur berikut:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama Saya</title>
</head>

<body>
    <h1>Selamat Datang</h1>
    <p>Saya sedang belajar HTML.</p>
</body>
</html>
```

### Penjelasan Kode

- `<!DOCTYPE html>` : Menentukan bahwa dokumen menggunakan HTML5.
- `<html>` : Elemen utama yang membungkus seluruh isi halaman.
- `<head>` : Berisi informasi halaman yang tidak ditampilkan langsung.
- `<title>` : Judul yang muncul pada tab browser.
- `<body>` : Tempat seluruh konten website ditampilkan.
- `<h1>` : Judul utama halaman.
- `<p>` : Digunakan untuk membuat paragraf.

## 4. Materi Dasar yang Perlu Dipelajari

Sebaiknya pelajari HTML secara bertahap:

### a. Heading dan Paragraf

Gunakan:

- `<h1>` sampai `<h6>`
- `<p>`

### b. Link dan Gambar

Gunakan:

- `<a>`
- `<img>`

### c. List

Gunakan:

- `<ul>`
- `<ol>`
- `<li>`

### d. Formulir

Pelajari:

- `<form>`
- `<input>`
- `<button>`

### e. Layout Dasar

Kenali penggunaan:

- `<div>`
- `<section>`
- `<header>`
- `<footer>`

## 6. Praktik Membuat Halaman Sederhana

Buat file baru dengan nama `profil.html`, lalu masukkan kode berikut:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Profil Singkat</title>
</head>

<body>

    <h1>Halo, Saya Bisma</h1>

    <p>
        Ini adalah halaman web sederhana yang dibuat menggunakan HTML.
    </p>

    <h2>Kegiatan Favorit</h2>

    <ul>
        <li>Bermain game</li>
        <li>Belajar desain web</li>
        <li>Mendengarkan musik</li>
    </ul>

    <p>
        Kunjungi akun GitHub saya
        <a href="https://github.com/">di sini</a>.
    </p>

</body>
</html>
```

---

## 7. Menjalankan File HTML

1. Simpan file dengan ekstensi `.html`.
2. Cari file yang sudah disimpan.
3. Klik dua kali file tersebut.
4. Browser akan menampilkan hasil halaman web yang telah dibuat.

---

## 8. Latihan Mandiri

Setelah berhasil menjalankan file HTML:

1. Ubah nama pada heading sesuai identitasmu.
2. Tambahkan minimal dua hobi baru.
3. Ubah judul tab browser.
4. Tambahkan satu gambar menggunakan tag `<img>`.
5. Simpan lalu tekan **F5** pada browser untuk melihat perubahan.

---

## Kesimpulan

HTML merupakan dasar dari pembuatan website. Dengan memahami struktur HTML dan berbagai tag dasar, kamu sudah memiliki fondasi untuk melanjutkan ke CSS dan JavaScript. Kunci utama dalam belajar HTML adalah sering mencoba dan mempraktikkan setiap materi yang dipelajari.
