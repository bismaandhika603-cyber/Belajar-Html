Bab 01

Apa itu HTML?

Kenalan dulu sebelum mulai coding

HTML singkatan dari HyperText Markup Language. Sederhananya, HTML adalah bahasa yang digunakan untuk membuat struktur halaman web. Setiap website yang pernah kamu buka — semuanya dibangun menggunakan HTML.

Kalau website diibaratkan rumah, maka HTML adalah rangka bangunannya (dinding, pintu, jendela). CSS adalah cat dan dekorasinya, dan JavaScript adalah peralatan elektronik di dalamnya.

📌 Penting
HTML bukan bahasa pemrograman — HTML adalah bahasa markup. Artinya kita memberi tanda (tag) pada teks untuk memberitahu browser bagaimana cara menampilkannya.

HTML bekerja dengan cara membungkus konten dalam tag. Tag ditulis dalam kurung sudut seperti <p>. Kebanyakan tag punya pembuka dan penutup:

contoh-tag.html 
<!-- Tag pembuka dan penutup -->
<p>Ini adalah sebuah paragraf.</p>
<h1>Ini adalah judul besar.</h1>

<!-- Tag kosong (tidak punya penutup) -->
<br>       <!-- garis baru -->
<hr>       <!-- garis pemisah -->
<img src="foto.jpg">  <!-- gambar -->

Tag juga bisa punya atribut — informasi tambahan yang ditulis di dalam tag pembuka:

atribut.html 
<!-- Format: <tag atribut="nilai"> -->
<a href="https://google.com">Buka Google</a>
<img src="foto.jpg" alt="Foto saya" width="300">
🧠 Kuis: Apa yang dimaksud "atribut" dalam HTML?
A Nama dari sebuah tag HTML
B Informasi tambahan yang ditulis di dalam tag pembuka
C Isi konten yang ditampilkan di browser
D Penutup dari sebuah tag
✅ Betul! Atribut memberikan informasi tambahan pada tag, seperti href pada link atau src pada gambar.
❌ Kurang tepat. Atribut adalah informasi tambahan yang ditulis di dalam tag pembuka, misalnya href="..." atau src="...".
Bab 02

Struktur Halaman HTML

Kerangka wajib yang ada di setiap file HTML

Setiap file HTML yang valid harus memiliki struktur dasar ini. Simpan file dengan ekstensi .html, lalu buka dengan browser.

index.html 
<!DOCTYPE html>
<html lang="id">

  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman Saya</title>
  </head>

  <body>
    <h1>Halo, Dunia!</h1>
    <p>Ini halaman web pertamaku.</p>
  </body>

</html>
Preview browser

Halo, Dunia!

Ini halaman web pertamaku.

Penjelasan setiap bagian:

  • <!DOCTYPE html> — Memberitahu browser bahwa ini file HTML5 (versi terbaru)
  • <html lang="id"> — Akar/pembungkus semua konten. lang="id" artinya bahasa Indonesia
  • <head> — Berisi informasi halaman yang tidak tampil di browser
  • <meta charset="UTF-8"> — Agar huruf seperti é, ñ, dan karakter Indonesia tampil benar
  • <meta name="viewport"> — Supaya halaman tampil baik di HP
  • <title> — Teks yang muncul di tab browser
  • <body> — Semua konten yang terlihat di browser ada di sini
💡 Tips
Gunakan VS Code dengan ekstensi Live Server — setiap kamu simpan file, browser otomatis reload tanpa refresh manual.
Latihan Buat halaman pertamamu

Ketik kode HTML di bawah ini, lalu klik Jalankan untuk melihat hasilnya:

  • Buat struktur HTML dasar
  • Tambahkan judul di <title>
  • Isi <body> dengan nama dan hobimu

Bab 03

Teks & Heading

Cara mengatur judul, paragraf, dan format teks

HTML punya 6 level heading, dari <h1> (terbesar) sampai <h6> (terkecil). Gunakan secara berurutan — <h1> hanya boleh ada satu per halaman sebagai judul utama.

teks.html 
<!-- Heading -->
<h1>Judul Utama (h1)</h1>
<h2>Subjudul (h2)</h2>
<h3>Sub-subjudul (h3)</h3>

<!-- Paragraf -->
<p>Ini paragraf biasa.</p>

<!-- Format teks -->
<p>Teks <strong>tebal</strong>, <em>miring</em>, dan <u>bergaris bawah</u>.</p>

<!-- Garis baru & pemisah -->
<p>Baris pertama.<br>Baris kedua (masih paragraf sama).</p>
<hr>
<p>Setelah garis pemisah.</p>

<!-- Kutipan -->
<blockquote>
  "Ilmu tanpa amal bagaikan pohon tanpa buah."
</blockquote>

Sub-subjudul (h3)

⚠️ Perhatian
Jangan pakai <h1> sampai <h6> hanya karena ukurannya. Gunakan sesuai hierarki konten. Ukuran bisa diatur dengan CSS nanti.
🧠 Kuis: Tag apa yang digunakan untuk membuat teks tebal yang punya makna penting?
A <b> — bold biasa
B <strong> — tebal + penanda penting
C <em> — miring
D <u> — bergaris bawah
✅ Tepat! <strong> lebih disarankan daripada <b> karena juga memberi makna semantik "penting" — bukan hanya tampilan tebal.
❌ Kurang tepat. <strong> adalah pilihan yang lebih baik karena memberi makna semantik "penting", bukan hanya sekadar tebal secara visual.
Bab 04

Link & Gambar

Menghubungkan halaman dan menampilkan gambar

Link (tautan) dibuat dengan tag <a>. Atribut href menentukan tujuan link:

link.html 
<!-- Link ke website lain -->
<a href="https://google.com">Buka Google</a>

<!-- Buka di tab baru -->
<a href="https://google.com" target="_blank">Buka di tab baru</a>

<!-- Link ke halaman lain di folder yang sama -->
<a href="tentang.html">Halaman Tentang</a>

<!-- Link ke bagian di halaman yang sama -->
<a href="#kontak">Ke bagian kontak</a>
<section id="kontak">...</section>
gambar.html 
<!-- Gambar dari file lokal -->
<img src="foto.jpg" alt="Foto pemandangan gunung">

<!-- Gambar dengan ukuran -->
<img src="logo.png" alt="Logo perusahaan" width="200">

<!-- Gambar dari internet -->
<img src="https://picsum.photos/400/200" alt="Gambar random">

<!-- Gambar yang bisa diklik -->
<a href="https://google.com">
  <img src="banner.jpg" alt="Klik untuk ke Google">
</a>
💡 Selalu isi atribut alt!
Atribut alt wajib ada di setiap gambar. Kegunaannya: (1) muncul jika gambar gagal dimuat, (2) dibaca oleh screen reader untuk pengguna tuna netra, (3) membantu SEO.
Latihan Buat kartu profil dengan link dan gambar

Buat halaman sederhana berisi:

  • Gambar (bisa pakai URL dari internet)
  • Nama sebagai heading
  • Link ke akun media sosialmu

Bab 05

List & Tabel

Menampilkan data dalam bentuk daftar dan tabel
list.html 
<!-- Unordered list (bullet) -->
<ul>
  <li>Nasi goreng</li>
  <li>Mie ayam</li>
  <li>Bakso</li>
</ul>

<!-- Ordered list (angka) -->
<ol>
  <li>Panaskan minyak</li>
  <li>Masukkan bawang</li>
  <li>Tambah nasi dan aduk</li>
</ol>

<!-- List di dalam list (nested) -->
<ul>
  <li>Makanan
    <ul>
      <li>Nasi goreng</li>
      <li>Mie ayam</li>
    </ul>
  </li>
  <li>Minuman</li>
</ul>
tabel.html 
<table border="1">

  <!-- thead = baris header -->
  <thead>
    <tr>
      <th>Nama</th>
      <th>Kota</th>
      <th>Nilai</th>
    </tr>
  </thead>

  <!-- tbody = baris data -->
  <tbody>
    <tr>
      <td>Budi</td>
      <td>Jogja</td>
      <td>95</td>
    </tr>
    <tr>
      <td>Siti</td>
      <td>Solo</td>
      <td>88</td>
    </tr>
  </tbody>

</table>

Singkatan yang perlu diingat: tr = table row (baris), th = table header (kolom header, otomatis tebal), td = table data (sel biasa).

Bab 06

Form & Input

Membuat form untuk interaksi dengan pengguna

Form digunakan untuk mengumpulkan data dari pengguna — mulai dari login, registrasi, hingga pencarian.

form.html 
<form action="/kirim" method="post">

  <!-- Input teks -->
  <label for="nama">Nama lengkap:</label>
  <input type="text" id="nama" name="nama" placeholder="Tulis namamu">

  <!-- Input email -->
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <!-- Dropdown -->
  <label for="kota">Kota:</label>
  <select id="kota" name="kota">
    <option value="">Pilih kota...</option>
    <option value="jogja">Yogyakarta</option>
    <option value="solo">Solo</option>
  </select>

  <!-- Radio button -->
  <p>Jenis kelamin:</p>
  <input type="radio" name="jk" value="L" id="laki">
  <label for="laki">Laki-laki</label>
  <input type="radio" name="jk" value="P" id="perempuan">
  <label for="perempuan">Perempuan</label>

  <!-- Textarea -->
  <label for="pesan">Pesan:</label>
  <textarea id="pesan" name="pesan" rows="4"></textarea>

  <!-- Tombol submit -->
  <button type="submit">Kirim</button>

</form>
💡 Tips label
Selalu hubungkan <label> dengan <input> menggunakan atribut for dan id yang sama. Ini membuat klik pada label otomatis fokus ke input-nya — dan penting untuk aksesibilitas.
Latihan Buat form kontak sederhana

Buat form dengan minimal: nama, email, dan pesan. Coba jalankan!

Bab 07

Div, Span & Elemen Semantik

Mengelompokkan konten dengan benar

<div> dan <span> adalah wadah generik. <div> bersifat block (mengambil seluruh lebar), sedangkan <span> bersifat inline (mengikuti aliran teks).

div-span.html 
<!-- div: blok container -->
<div style="background: #f0f0f0; padding: 16px;">
  <h2>Kartu Profil</h2>
  <p>Ini konten di dalam div.</p>
</div>

<!-- span: inline styling -->
<p>
  Harga: <span style="color: red; font-weight: bold;">DISKON 50%</span> hari ini!
</p>

HTML5 memperkenalkan elemen semantik — tag yang namanya menggambarkan isinya. Ini lebih baik daripada menulis semua dalam <div>:

semantik.html 
<header>
  <nav>
    <a href="/">Beranda</a>
    <a href="/tentang">Tentang</a>
  </nav>
</header>

<main>
  <article>
    <h1>Judul Artikel</h1>
    <p>Isi artikel...</p>
  </article>

  <aside>
    <h3>Artikel Terkait</h3>
  </aside>
</main>

<footer>
  <p>© 2025 Website Saya</p>
</footer>

Elemen semantik yang sering dipakai:

  • <header> — Bagian atas halaman/section
  • <nav> — Menu navigasi
  • <main> — Konten utama halaman
  • <article> — Konten mandiri (blog post, berita)
  • <section> — Bagian/kelompok konten bertema
  • <aside> — Konten sampingan (sidebar)
  • <footer> — Bagian bawah halaman
💡 Kenapa semantik penting?
Elemen semantik membantu mesin pencari memahami struktur halaman (SEO lebih baik), membuat kode lebih mudah dibaca, dan meningkatkan aksesibilitas untuk pengguna dengan kebutuhan khusus.
🧠 Kuis: Apa perbedaan utama antara <div> dan <span>?
A <div> untuk teks, <span> untuk gambar
B <div> adalah block element, <span> adalah inline element
C Keduanya sama persis, hanya beda nama
D <span> lebih baru dari <div>
✅ Betul! <div> memakan satu baris penuh (block), sedangkan <span> mengikuti aliran teks (inline) sehingga tidak memecah paragraf.
❌ Kurang tepat. Perbedaan utamanya adalah perilaku layout: <div> = block (satu baris sendiri), <span> = inline (menyatu dalam teks).
